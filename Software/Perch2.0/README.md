#Perch 2.0 Integration Studies

This directory contains legacy folders (`V1/`, `V2/`) analyzing and validating embeddings specifically powered by the Perch 2.0 sound event detection model.

## Overview

The models enclosed within explore the raw extraction limits of Perch 2.0, primarily looking into how well generalized bird/environmental sound embeddings cluster inside unsupervised anomaly classification schemes.

> **Note**: For the more comprehensive, end-to-end data processing pipeline utilizing the Perch model, refer to [`../anomaly-data-loader`](../anomaly-data-loader).

## Usage Requirements

To fully utilize these scripts, you will likely need the ONNX-formatted Perch model.

*   📥 [**Download `perch_v2.onnx` here**](https://huggingface.co/justinchuby/Perch-onnx/resolve/main/perch_v2.onnx?download=true)
