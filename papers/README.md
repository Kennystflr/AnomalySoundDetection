# 📚 Research Literature

This directory contains key research papers that form the theoretical foundation and state-of-the-art context for our Anomaly Sound Detection projects.

## Included Literature

*   **`2512.03219v1.pdf` & `Ferreiraetal2025.pdf`**: Recent advancements in anomaly detection architectures and methodologies.
*   **`Acoustic-Anomaly-Detection-of-Machinery-using-Autoencoder-based-Deep-Learning.pdf`**: Implementing autoencoders to capture baseline mechanical sounds and reject irregular vibrations or anomalous noises.
*   **`Anomalib_A_Deep_Learning_Library_for_Anomaly_Detection.pdf`**: Documentation and foundation on Anomalib, highlighting standard library design for visual and audio anomalies.
*   **`Efficient_Anomaly_Detection_Algorithm_for_Heart_Sound_Signal.pdf`**: Adapting anomaly detection pipelines for biomedical audio signals.
*   **`Enhancing_Sound-Based_Anomaly_Detection_Using_Deep_Denoising_Autoencoder.pdf`**: Utilizing Denoising Autoencoders to separate baseline sounds from latent anomalies effectively.
*   **`anomalous-sound-event-detection-based-on-wavenet.pdf`**: An approach involving WaveNet algorithms for pinpointing sequential anomalous audio events.

These papers are actively used as reference points for architecture design decisions, especially in formulating semi-supervised models within the `Software/ar_beats` and `Software/anomaly-data-loader` projects.
